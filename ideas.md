## 方案一：Cosmic Mysticism（宇宙神秘主義）

**設計運動**：參考 Active Theory 的沉浸式數位體驗，結合東方神秘學的視覺語彙，創造一種「宇宙殿堂」般的瀏覽體驗。

**核心原則**：
1. 深邃的宇宙空間感——以極深藍黑色為基底，營造無垠星空的沉浸感
2. 粒子與光的敘事——用浮動粒子和光暈效果暗示靈性能量的流動
3. 儀式感的排版——大型等寬字體標題搭配優雅的襯線體內文，如同神諭般莊嚴
4. 漸進式揭示——內容隨滾動逐步顯現，模擬占卜時翻牌的期待感

**色彩哲學**：以深空黑（#06060f）為畫布，靛藍（#1a1a4e）為深度層次，金色（#c9a84c）作為神聖的強調色，青色（#4ecdc4）作為靈性引導色。金色代表宇宙智慧，青色代表療癒能量。

**佈局範式**：全螢幕垂直滾動敘事，每個區段佔滿視窗高度。左側固定垂直導航文字，右側浮動裝飾元素。商品展示採用大型文字列表搭配懸浮預覽圖的方式。

**標誌性元素**：
1. Canvas 粒子星空背景——持續飄浮的金色微粒
2. 虹彩光暈裝飾——模擬 Active Theory 的 3D 金屬環效果
3. 文字描邊效果——商品名稱使用描邊空心字體

**互動哲學**：滑鼠移動觸發粒子軌跡，滾動觸發區段淡入，懸停商品時展開詳細資訊。一切互動都緩慢而優雅，如同冥想般的節奏。

**動畫**：使用 Framer Motion 實現平滑的區段進入動畫（淡入 + 上移），粒子系統使用 Canvas API 繪製持續飄浮的星塵效果，文字採用逐字顯現動畫。

**字體系統**：標題使用 Space Mono（等寬字體，全大寫，字距加寬），副標題使用 Cormorant Garamond（優雅襯線體），內文使用 Noto Sans TC（清晰的中文無襯線體）。

## 方案二：Ethereal Scroll（空靈卷軸）

**設計運動**：融合日本侘寂美學與數位藝術，以「展開卷軸」為核心隱喻，創造一種安靜而深邃的閱讀體驗。

**核心原則**：
1. 留白即敘事——大量黑色空間中點綴微光，讓內容如星辰般珍貴
2. 水墨般的過渡——區段之間使用漸變融合，如同水墨在宣紙上暈染
3. 東方韻律——以不對稱構圖和黃金比例安排元素位置
4. 呼吸感的節奏——內容間距寬鬆，給予觀者思考的空間

**色彩哲學**：墨黑（#0a0a0a）為底色，月白（#e8e4df）為文字色，朱砂紅（#c4463a）為點綴，藏青（#1a3a5c）為過渡色。色彩取自傳統東方繪畫，但以現代數位方式呈現。

**佈局範式**：水平與垂直混合滾動，模擬卷軸展開的體驗。內容偏左或偏右排列，從不置中，創造動態的視覺流動。

**標誌性元素**：
1. 水墨粒子效果——粒子如墨點般在深色背景中飄散
2. 書法風格的裝飾線條——細長的金色線條劃分區段
3. 印章式的品牌標記——Logo 以印章形式出現在角落

**互動哲學**：極簡的互動設計，僅在必要時給予回饋。滾動是主要的互動方式，每次滾動都像翻開新的一頁。

**動畫**：緩慢的淡入動畫（1.2 秒），元素從下方浮起。背景粒子以極慢速度漂移。頁面切換使用交叉淡出效果。

**字體系統**：標題使用 Zen Antique（日式襯線體），中文標題使用 Noto Serif TC，內文使用 Noto Sans TC light weight。

## 方案三：Astral Portal（星界入口）

**設計運動**：受 Brutalist Web Design 與 Sci-Fi UI 啟發，以「星際控制台」為視覺主題，將占卜工具包裝為未來科技般的體驗。

**核心原則**：
1. 網格即結構——以精密的網格線和座標系統組織所有元素
2. 數據視覺化的美學——將品牌資訊以儀表板、圖表、座標的方式呈現
3. 機械感的精確——所有動畫都有明確的起止點，如同機器運轉
4. 層疊的資訊架構——多層半透明面板重疊，暗示多維空間

**色彩哲學**：純黑（#000000）為虛空，螢光綠（#00ff88）為數據流，冷白（#f0f0f0）為資訊文字，深紫（#2a0845）為能量場。這是一個將神秘學數位化的色彩系統。

**佈局範式**：CSS Grid 驅動的多面板佈局，類似太空站控制台。固定的頂部狀態列顯示「座標」（即導航），主內容區域分為多個可滾動的面板。

**標誌性元素**：
1. 掃描線效果——CRT 螢幕般的水平掃描線覆蓋
2. 座標網格——半透明的網格線覆蓋整個頁面
3. 閃爍的游標——等寬字體文字帶有打字機效果

**互動哲學**：精確而即時的回饋，如同操作精密儀器。懸停時顯示座標資訊，點擊時有「啟動」般的動畫效果。

**動畫**：打字機效果的文字顯現，面板滑入動畫，掃描線持續移動，數據流般的粒子效果。

**字體系統**：全站使用 JetBrains Mono（等寬字體），中文使用 Noto Sans Mono CJK TC，不使用襯線體，強調機械感和精確性。
